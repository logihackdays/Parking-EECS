# Parking-EECS
停車遊戲 Parking Game

<h3>組成</h3>
<ol>
<li>停車遊戲主體：以一個Windows Form呈現遊戲畫面</li>
<li>鍵盤LED</li>
</ol>

<h3>Feature</h3>
<p>在遊戲中打方向燈的時候，鍵盤LED會組成箭頭的符號</p>

<h3>玩法</h3>
<ul>
<li>D – 油門(前進/後退)</li>
<li>左右鍵 – 轉彎</li>
<li>A/S – 左/右轉燈	</li>
<li>shift – 換檔(D/R)</li>
</ul>

<h3>其他構想</h3>
<ul>
<li>用方向盤、油門、煞車控制遊戲中的車子</li>
<li>第二螢幕在倒車的時候會警示距離後方障礙物有多遠</li>
</ul>

<h3>開發問題</h3>
<p>無法用SDK取到方向盤或是其他裝置的值，有可能和Windows Form有關</p>
